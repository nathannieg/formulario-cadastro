## Sobre

O objetivo do desafio era desenvolver um formulário de cadastrado a partir de um layout desenvolvido na ferramenta Figma, contendo campos:

- De texto curto e longo;
- E-mail;
- Select;
- Checkbox;
- Data;
- Hora.
